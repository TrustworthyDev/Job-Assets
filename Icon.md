### Icon Creation Specification

# Canvas
- Square canvas (e.g., 1024×1024).
- Fully transparent background.

# Outer Frame
- Rounded square stroke only (no fill).
- Thick border placed close to the canvas edges so the icon uses maximum visual area.
- Apply a multi-color gradient around the border (random of cyan → blue → purple → pink → red → orange → yellow → green).

# Text
- Place 3-letter text (Ros) centered horizontally, but slightly lowered vertically inside the frame.
- Font: bold sans-serif.
- Text fill: pure white.
- Text outline: same gradient colors as the frame.

# Layout
- Text should occupy about 60–70% of the icon height.
- Lower the text slightly (around 5–10% downward) so it sits a bit closer to the bottom of the square.
- No extra elements.

# Export
- Transparent .ico file including sizes: e.g. 16, 24, 32, 48, 64, 128, 256 px.
